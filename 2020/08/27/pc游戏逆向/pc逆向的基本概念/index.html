pc逆向

pc逆向的基本概念

DLL文件

  • dynamics link lib 动态连接库，可以认为是程序的基本功能包。360，火绒剑都可以查看进程和调用的DLL。
  • 感染共享的DLL 获取其他软件的权限
  • DLL 本质上是一个代码库，程序需要时候加载不需要不加载windows以这种方式节省程序的内存占用。

    常用的语言和工具

  • winshark
  • c++ vs2017
  • 易语言

    安全术语

  • r0-r3,r0是系统 r3是应用

    windows系统相关

  • windows系统是一个服务中心集合。
  • API is app programming interface for short。
  • 应用层基本使用win32的API，r0级别会使用win64的API
